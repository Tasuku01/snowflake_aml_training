## 1_データベースの作成 

`CREATE DATABASE AML_TRAINING;`

## 2_スキーマの作成

`CREATE SCHEMA AML;`

## 3_テーブルの作成
`
CREATE TABLE accounts (
    account_id NUMBER,
    customer_id NUMBER,
    open_date DATE
);
`

`
CREATE TABLE alert_rules (
    alert_type STRING,
    threshold_type STRING,
    threshold_value NUMBER,
    description STRING
);
`

`
CREATE TABLE aml_alerts (
    alert_id NUMBER,
    customer_id NUMBER,
    alert_type STRING,
    alert_date DATE
);
`

`
CREATE TABLE customer_kyc (
    customer_id NUMBER,
    occupation STRING,
    annual_income NUMBER,
    pep_flag NUMBER
);
`

`
CREATE TABLE customers (
    customer_id NUMBER,
    customer_name STRING,
    country STRING,
    risk_level STRING
);
`

`
CREATE TABLE transactions (
    txn_id NUMBER,
    account_id NUMBER,
    txn_date DATE,
    amount NUMBER,
    txn_type STRING,
    destination_country STRING
);
`

## 4_csvファイルのアップロード

以下の6つのファイルをアップロードする

・accounts.csv

・alert_rules.csv

・aml_alerts.csv

・customers.csv

・customer_kyc.csv

・transactions.csv


## 5_SQLを実行する前の準備

以下のコマンドを実行してDBを選択する

`USE DATABASE AML_TRAINING;`

続けて以下のコマンドを実行してスキーマを選択する

`USE SCHEMA AML;`
